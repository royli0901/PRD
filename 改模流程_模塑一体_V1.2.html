概
流程概述与核心逻辑

本流程适用于模塑一体业务中发生的模具设变（改模）管理，覆盖从设变需求发起到 SO 下达、PO 跟进的完整闭环。

📌
SO 与 PO 的关系（V1.2 核心修正）：自改和客改均需下达 SO，SO 是精密模具中心开项目号、启动改模作业的内部授权凭证。区别在于：自改无需客户 PO；客改需要客户 PO，但 SO 先于 PO 下达，SO 同时标记"待收 PO"状态，自动提醒客户经理跟进。
🎯
效率目标：明确六个角色职责边界，SO先行保证改模不等待，避免因收款流程拖延生产。
🔒
管控目标：客改收费归属书面确认，SO 上标记 PO 状态，确保追款动作内嵌流程不遗漏。
①
六个角色与职责边界
角色部门核心职责关键输出
注塑BU模具工程师 注塑BU 流程发起人。收集产线设变需求，通过 OA 发起设变申请。不参与自改/客改判断，避免既做裁判又做运动员。 设变需求单（OA，含模号、产品名称、改动内容）
注塑项目工程师 注塑BU 归因裁判 + SO下达。在精密模具方案出具后判断自改/客改（最终结论）。自改后直接下达 SO；客改确认收费归属后，按金额分流。 归因结论、SO 下达
精密模具项目工程师 精密模具中心 通过 OA 接收设变需求，协调模具工程师制定改模方案。可直接与客户对接技术方案，但收费确认须留书面记录并通知注塑项目工程师。凭 SO 开项目号启动改模作业。 改模方案（DFM、工时、造价），改模执行
精密模具工程师 精密模具中心 协同精密模具项目工程师评估改模技术可行性，提供工时估算和技术参数。 工时估算、技术方案细节
成本工程师 报价中心 仅在客改且金额 ≥ 1万时介入。基于精密模具造价估算，出具成本分析面单并同步 CRM，面单完成后由注塑项目工程师下达 SO。 成本分析面单（CRM 同步）
客户经理 营销中心 仅在客改路径参与。SO 下达后系统自动提醒，跟进客户 PO。< 1万直接催收；≥ 1万基于成本面单与客户确认报价后催收 PO。自改路径不参与。 客户 PO 确认
②
自改 / 客改归因标准
⚠️
判断时机：归因判断须在精密模具方案出具之后进行，以造价估算为客观依据。注塑项目工程师不得在方案未出前主观判定，判断结论为最终结论，不可申诉。
🔵 自改情形（公司内部承担费用）
  • 模具正常使用损耗（顶针断裂、镶件磨损）
  • 模具制造时的工艺失误（尺寸偏差、粗糙度不达标）
  • 模具设计缺陷（走胶不顺、困气、缩水）
  • 维护保养不当导致的损坏
  • 公司内部试模过程中造成的损伤
  • DFM 阶段公司评估失误，导致后期需改模
🟠 客改情形（费用由客户承担）
  • 客户产品设计变更（ECN）——尺寸、结构、壁厚调整
  • 客户更改材料规格（影响收缩率或工艺参数）
  • 客户更改外观要求（纹路、光泽度、颜色）
  • 客户更改功能要求（配合关系、装配结构）
  • 客户更改标准或认证要求（标识、材料牌号）
  • 客户要求增减模穴数量
⚡ 灰色地带处理规则
情形判定规则
客户投诉产品不良，但根因是模具问题自改 问题根因决定费用归属，不因客户投诉改变
DFM 阶段客户已签字确认，量产后不满意客改 以 DFM 书面签字记录为准，已确认即视为客改
模具到期需翻新，合同无寿命约定待议 双方协商；建议后续新合同加入模具寿命条款
客户要求提升性能，但模具本身亦有缺陷拆分 缺陷修复部分自改，性能提升部分客改，分别核算
③
三条路径详细步骤
🔵 路径A：自改（任意金额）
步骤操作执行角色输出
S1通过 OA 发起设变需求注塑BU模具工程师设变需求单
S2OA 接收，协调制定改模方案（DFM、工时、造价）精密模具项目工程师 + 模具工程师改模方案文件
S3审核方案，裁定为自改注塑项目工程师归因结论（最终，不可申诉）
S4直接下达 SO，费用公司自担，无需客户 PO，客户经理不参与注塑项目工程师SO（内部授权凭证）
S5凭 SO 开项目号，安排改模作业精密模具中心改模完成
🟠 路径B：客改（共同步骤）
步骤操作执行角色输出/要求
C1–C3同自改 S1–S3同上同上
C4裁定为客改，与客户确认收费归属书面必留注塑项目工程师（主责）或精密模具项目工程师（效率）邮件/微信截图，明确"客户同意付费"
C5判断改模金额（以方案造价估算为准）注塑项目工程师金额分流判断
客改 · 金额 < 1万元
  • 直接下达 SO（不经过成本工程师）
  • SO 标记"待收 PO"，系统提醒客户经理
  • 精密模具中心凭 SO 开项目号，启动改模
  • 客户经理同步跟进客户 PO（改模不等 PO）
客改 · 金额 ≥ 1万元
  • CRM 自动生成报价需求（请求表 + 版本号）
  • 成本工程师出具成本分析面单，同步 CRM
  • 面单完成后下达 SO，SO 标记"待收 PO"
  • 精密模具中心凭 SO 开项目号，启动改模
  • 客户经理基于成本面单跟进客户 PO
💡
SO 先行的设计逻辑：SO 是生产授权凭证，精密模具中心必须持有 SO 才能开项目号启动改模。SO 上的"待收 PO"状态同时充当系统级跟进提醒，将追款动作内嵌进流程，不依赖人工记忆。改模作业与 PO 跟进并行推进，效率和管控两不误。
④
关键规则与风险提示
📝
书面记录强制要求：不论是注塑项目工程师还是精密模具项目工程师与客户沟通，以下两件事必须留书面记录：① 客户明确确认"费用由客户承担"；② 客户对改模金额有书面回复（≥1万时须对报价有明确书面确认）。口头确认不作为追款依据。
⚡
精密模具项目工程师直接对客时的边界：负责技术方案沟通，收费归属和金额属于商务事项，须同步通知注塑项目工程师，不得单独以口头形式向客户承诺价格，避免与成本工程师核算结果出入造成内部被动。
🔮
待完善事项：目前模具合同未约定模具寿命，建议后续新合同加入模具寿命条款（以模次或年限），从根本上消除翻新费用归属争议。

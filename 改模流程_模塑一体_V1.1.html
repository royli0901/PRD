概
流程概述与目的

本流程适用于模塑一体业务中发生的模具设变（改模）管理，覆盖从设变需求发起到 SO 下达的完整闭环。核心目标：

🎯
效率目标：明确六个角色的职责边界，避免推诿，保证改模响应速度。
🔒
管控目标：确保客改费用有书面依据，SO 下达前完成收费归属确认，保障公司收款权益。
①
六个角色与职责边界
角色部门核心职责关键输出
注塑BU模具工程师 注塑BU 流程发起人。收集产线设变需求，通过 OA 发起设变申请。不参与自改/客改判断，避免既做裁判又做运动员。 设变需求单（OA，含模号、产品名称、改动内容、附件PPT）
注塑项目工程师 注塑BU 归因裁判。在精密模具方案出具后，依据造价估算和改动原因，判断本次改模归属自改或客改。判断结论为最终结论，不可申诉。 自改/客改归因结论（书面记录）
精密模具项目工程师 精密模具中心 通过 OA 接收设变需求，协调模具工程师制定改模方案。可直接与客户对接技术方案（基于效率），但收费确认须留书面记录并通知注塑项目工程师。 改模方案（DFM、工时、改模时间、造价估算）
精密模具工程师 精密模具中心 协同精密模具项目工程师评估改模技术可行性，提供工时估算和技术参数。 工时估算、技术方案细节
成本工程师 报价中心 仅在金额 ≥ 1万时介入。基于精密模具造价估算，出具成本分析面单并同步 CRM。 成本分析面单（CRM 同步）
客户经理 营销中心 金额 < 1万：直接与客户确认 PO，下达 SO。金额 ≥ 1万：基于成本面单与客户谈判并跟进 PO。 客户 PO 确认、SO 下达
②
自改 / 客改归因标准
⚠️
判断时机：归因判断须在精密模具方案出具之后进行，以造价估算为客观依据。注塑项目工程师不得在方案未出前主观判定，避免缺乏依据的随意归因。
🔵 自改情形（公司内部承担费用）
  • 模具正常使用损耗（顶针断裂、镶件磨损）
  • 模具制造时的工艺失误（尺寸偏差、粗糙度不达标）
  • 模具设计缺陷（走胶不顺、困气、缩水）
  • 维护保养不当导致的损坏
  • 公司内部试模过程中造成的损伤
  • DFM 阶段公司评估失误，导致后期需改模
🟠 客改情形（费用由客户承担）
  • 客户产品设计变更（ECN）——尺寸、结构、壁厚调整
  • 客户更改材料规格（影响收缩率或工艺参数）
  • 客户更改外观要求（纹路、光泽度、颜色）
  • 客户更改功能要求（配合关系、装配结构）
  • 客户更改标准或认证要求（标识、材料牌号）
  • 客户要求增减模穴数量
⚡ 灰色地带处理规则
情形判定规则
客户投诉产品不良，但根因是模具问题自改 问题根因决定费用归属，不因客户投诉改变
DFM 阶段客户已签字确认，量产后不满意客改 以 DFM 书面签字记录为准，已确认即视为客改
模具到期需翻新，合同无寿命约定待议 双方协商；建议后续新合同加入模具寿命条款
客户要求提升性能，但模具本身亦有缺陷拆分 缺陷修复部分自改，性能提升部分客改，分别核算
③
两条主路径详细步骤
🔵 路径A：自改
步骤操作执行角色输出
S1通过 OA 发起设变需求注塑BU模具工程师设变需求单（含模号、改动内容）
S2OA 接收需求，协调制定改模方案精密模具项目工程师 + 模具工程师改模方案（DFM、工时、造价）
S3审核方案，裁定为自改注塑项目工程师归因结论（最终，不可申诉）
S4安排内部改模，费用自担精密模具中心改模完成，无需客户 PO 及 SO
🟠 路径B：客改
步骤操作执行角色输出/要求
C1–C3同自改 S1–S3同上同上
C4裁定为客改，与客户确认收费归属书面必留注塑项目工程师（主责）
或 精密模具项目工程师（效率）		邮件/微信截图，明确"客户同意付费"
C5判断改模金额（以方案造价估算为准）注塑项目工程师金额判断结论
金额 < 1万元
  • 客户经理直接与客户沟通改模费用
  • 客户确认后下达 PO
  • 公司下达 SO，安排改模
  • 成本工程师不介入
金额 ≥ 1万元
  • CRM 自动生成报价需求（请求表 + 版本号）
  • 成本工程师出具成本分析面单，同步 CRM
  • 客户经理基于面单与客户谈判确认报价
  • 客户下达 PO → 公司下达 SO → 安排改模
④
关键规则与风险提示
📝
书面记录强制要求：不论是注塑项目工程师还是精密模具项目工程师与客户沟通，以下两件事必须留书面记录：① 客户明确确认"费用由客户承担"；② 客户对改模金额有书面回复（≥1万时须对报价有明确书面确认）。口头确认不作为追款依据。
⚡
精密模具项目工程师直接对客时的边界：负责技术方案沟通，收费归属和金额属于商务事项，须同步通知注塑项目工程师，不得单独以口头形式向客户承诺价格，避免与成本工程师核算结果出入造成内部被动。
🔮
待完善事项：目前模具合同未约定模具寿命，建议后续新合同加入模具寿命条款（以模次或年限），从根本上消除翻新费用归属争议。
